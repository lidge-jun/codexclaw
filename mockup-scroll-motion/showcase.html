AI 모션 그래픽 파이프라인

이미지 생성에서 웹 스크롤 액션까지 4단계

01

키프레임 이미지 생성

GPT Image, Grok Aurora, Midjourney 등으로 시작 프레임을 생성합니다. 직접 촬영하거나 제작한 이미지도 사용 가능합니다. ima2 gen 명령어로 CLI에서 바로 생성할 수 있습니다.

02

비디오 모션화

Grok Video 1.5로 이미지를 모션화합니다. 최대 1080p, 15초, ref 이미지 7장까지 지원. ima2 video 명령어로 image-to-video 변환이 가능합니다.

03

프레임 시퀀스 추출

ffmpeg로 비디오를 개별 WebP 프레임으로 분할합니다. 프레임 수, 해상도, 포맷을 용도에 맞게 조절합니다. Apple 방식의 체크포인트 로딩으로 최적화합니다.

04

웹 스크롤 액션 구현

Canvas API로 프레임 시퀀스를 스크롤 진행률에 매핑하거나, CSS scroll-driven animation으로 sprite sheet를 연동합니다. sticky 컨테이너 안에서 재생됩니다.

네이티브 CSS 시대

animation-timeline: scroll()로 JS 스크롤 리스너 없이 컴포지터 스레드에서 실행. Tokopedia 사례: CPU 50% -> 2%.
