Scroll-Driven Frame Sequence

AI 생성 이미지를 비디오로 모션화하고, 프레임 시퀀스로 분할해 스크롤에 연동하는 프로토타입

스크롤해서 재생
001 / 120
GPT Image로 생성한 키프레임에서 시작
Grok Video 1.5로 모션화된 중간 프레임
스크롤 위치에 프레임이 1:1 매핑

파이프라인 워크플로우

1
이미지 생성

키프레임 이미지 생성

GPT Image, Grok Aurora, Midjourney 등으로 시작 프레임을 생성하거나, 직접 제작한 이미지를 사용합니다.

ima2 gen "floating product on dark gradient" --quality high
2
모션화

이미지 → 비디오 변환

생성된 이미지를 Grok Video 1.5, Runway, Kling 등으로 모션화합니다. 직접 촬영하거나 편집한 비디오도 사용 가능합니다.

ima2 video "slow orbit with light shift" \ --ref keyframe.png --duration 10 \ --resolution 1080p --aspect-ratio 16:9
3
프레임 분할

비디오 → 프레임 시퀀스 추출

ffmpeg로 비디오를 개별 프레임으로 분할합니다. 프레임 수와 포맷은 용도에 맞게 조절합니다.

ffmpeg -i motion.mp4 -vf "fps=24" frames/%04d.webp
4
웹 구현

CSS Scroll-Driven 또는 Canvas 시퀀스

CSS animation-timeline: scroll()로 스프라이트 시트를 스크롤에 연동하거나, Canvas API로 프레임 시퀀스를 직접 그립니다.
