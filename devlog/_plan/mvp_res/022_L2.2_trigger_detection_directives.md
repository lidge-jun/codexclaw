## Goal (one slice)
Map explicit English and Korean prompts to the shipped IPABCD directive text.

## Why now / dependencies
After hook IO is safe, cxc needs deterministic phase selection. L2.3 uses this result to activate
orchestration and decide whether to inject a full directive or a short stage header.

## Scope (decision-complete)
- Files added/edited:
  - `plugins/codexclaw/components/pabcd-state/src/hook.ts`
  - `plugins/codexclaw/components/pabcd-state/test/hook.test.ts`
- Functions shipped:
  - `detectTrigger(prompt)`
  - `phaseDirective(phase)`
  - `interviewDirective()`
  - `buildStageHeader(phase)`
- Explicit trigger mapping:
  - `interview`, `orchestrate i`, Korean interview trigger -> `I`
  - `orchestrate p`, `plan this`, Korean plan trigger -> `P`
  - `orchestrate a`, `audit this`, Korean audit trigger with strong action marker -> `A`
  - `orchestrate b`, `build this`, Korean build trigger -> `B`
  - `orchestrate c`, `check this`, Korean check trigger -> `C`
- Directive text shipped:
  - `[codexclaw: INTERVIEW]`
  - `[codexclaw: PLAN]`
  - `[codexclaw: AUDIT]`
  - `[codexclaw: BUILD]`
  - `[codexclaw: CHECK]`
  - `[codexclaw: DONE]`
- Must-NOT-Have:
  - No detection from ordinary non-trigger prompts.
  - No bare Korean thanks word audit trigger, because it collides with thanks.
  - No goal-mode branch in this loop.

## IPABCD micro-cycle
- I (if interview-bearing): explicit interview prompts inject the `I` directive only.
- P: trigger order checks interview first so `orchestrate i` wins over plan text.
- A: C-gate audit found the Korean thanks false positive.
- B: audit trigger was anchored to strong Korean action markers documented in `hook.ts`.
- C: tests cover English, Korean, interview precedence, non-triggers, and everyday Korean thanks.
- D: done = phase directives are deterministic and avoid the known Korean false positive.

## Acceptance (1-3 testable criteria)
- A prompt containing both interview and plan language maps to `I`.
- Ordinary Korean thanks forms do not map to `A`.
- Natural Korean trigger forms with particles or suffixes still map to the intended phase.
