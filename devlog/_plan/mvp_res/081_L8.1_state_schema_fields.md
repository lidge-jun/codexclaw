## Goal (one slice)
Add the concrete interview tracker fields to the existing PABCD state without replacing any live Phase 1 fields.

## Why now / dependencies
L1 state persistence and L2 directive injection must already be done. This sub-pass gives L8.2 and L8.3 a typed tracker to read, reconstruct, and bound.

## Scope (decision-complete)
- Files to add/edit: `plugins/codexclaw/components/pabcd-state/src/state.ts`, `plugins/codexclaw/components/pabcd-state/test/state.test.ts`.
- Extend, do not replace, the live `State` interface; preserve fields such as phase, sessionId, slug, updatedAt, flags, supersededBy, injectedTurns, and existing orchestration markers.
- Add `interview: InterviewTracker | null` to `State`, `defaultState()`, and the strict `readState()` reconstruction literal.
- Add `Dimension = goal | constraint | success | ontology` and `DIMENSIONS` in that order.
- Add `DimensionLevel = low | mid | high | max` and `DimensionScore` with `level`, `known`, `unknown`, and `confidence`.
- Add `ContradictionSeverity = low | medium | high`, `Contradiction`, `Assumption`, and `InterviewTracker`.
- `Assumption.recorded` means the assumption will be emitted to `## OPEN ASSUMPTIONS`; readiness does not require assumptions to be empty.
- Fresh sessions must read `interview: null`; `defaultInterview()` must initialize all four dimensions at low and empty arrays.
- Must-NOT-Have: no hook directive changes, no Mind prompts, no freeze manifest, no goal-mode hard deny.

- P: Patch type declarations and default state before reconstruction helpers so tests can assert shape stability.

## Acceptance (1-3 testable criteria)

2. `defaultInterview()` returns all four dimensions, empty assumptions, empty contradictions, rounds 0, and ready false.
3. A full tracker persists and reads back without unknown-key passthrough.
