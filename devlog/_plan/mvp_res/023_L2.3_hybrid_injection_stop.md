# L2.3 (Decade 023) -- Hybrid Injection and Passive Stop

Status: DONE
Cluster: 1 - Phase: 1 - Shorthand: cxc

## Goal (one slice)
Apply directive injection to UserPromptSubmit turns while keeping Stop passive in Phase 1.

## Why now / dependencies
L2.1 provides hook IO and L2.2 provides the phase directive. This sub-pass joins them to session
state from L1 and defines the runtime behavior later verified by L7 S2.

## Scope (decision-complete)
- Files added/edited:
  - `plugins/codexclaw/components/pabcd-state/src/hook.ts`
  - `plugins/codexclaw/components/pabcd-state/src/state.ts`
  - `plugins/codexclaw/components/pabcd-state/src/cli.ts`
  - `plugins/codexclaw/hooks/user-prompt-submit-checking-pabcd-trigger.json`
  - `plugins/codexclaw/hooks/stop-checking-pabcd-continuation.json`
- Functions and state:
  - `handleUserPromptSubmit(payload)`
  - `handleStop(payload)`
  - `injectedTurns` is bounded to 50 turns.
  - `lastInjectedPhase` records the last full directive phase.
  - `orchestrationActive` fail-closes non-trigger sessions.
- Three runtime modes:
  - explicit trigger: activate orchestration and inject full directive.
  - active and phase changed: inject full directive for current state phase.
  - active and same phase: inject short `[codexclaw -- X: LABEL]` stage header.
- Must-NOT-Have:
  - No injection in fresh non-trigger sessions.
  - No double injection for the same `(session_id, turn_id)`.
  - No Stop state write, ledger write, or phase auto-advance.

## IPABCD micro-cycle
- I (if interview-bearing): explicit interview triggers activate orchestration and inject the full I text.
- P: use hybrid 018.3 behavior instead of one-shot trigger-only injection.

- C: hook tests prove idempotency, session independence, fail-closed behavior, and no Stop ledger.
- D: done = runtime injection is deterministic and Stop remains a documented no-op.

## Acceptance (1-3 testable criteria)
- Fresh non-trigger prompt returns empty output and writes no state.
- Re-fired hook with the same `turn_id` returns empty output after the first injection.
- Stop always returns empty output and creates no `.codexclaw/ledger.jsonl`.
