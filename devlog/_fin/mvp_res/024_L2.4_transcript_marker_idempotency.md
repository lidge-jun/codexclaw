## Goal (one slice)
Harden UserPromptSubmit idempotency beyond the local `injectedTurns` flag, which is vulnerable after
compaction (turn_id reuse/divergence). Port omo's two transcript-grounded guards:
- transcript-marker scan: if the recent transcript tail already carries a codexclaw stage marker for
  the CURRENT phase, skip re-injection (dedup survives turn_id churn + compaction).
- context-pressure suppression: if the transcript tail / prompt shows a compaction-recovery marker,
  stay silent (don't pile directive context onto an already-pressured window).

## Why now / dependencies
L2 is the directive-injection surface; every later loop's per-turn injection inherits its dedup. The
shipped guard is local-flag only (007 R-11). This is an L2 reconciliation of the L1-L7 cluster.
