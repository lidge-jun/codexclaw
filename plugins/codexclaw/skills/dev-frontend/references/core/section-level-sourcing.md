# Section-Level Sourcing Map

Use this reference when a specific landing-page section needs curated examples
before composition begins. Counts are the observed counts from the 2026-07-14
research snapshot; gallery inventories change, so treat them as dated signals.

> **Sourcing boundary (FE-SECTION-SOURCE-01, STRICT):** This file is a sourcing
> pointer, not a composition rule. Examples may broaden the option set, but
> `layout-discipline.md` remains authoritative for section grammar, repetition,
> content limits, containment, and responsive transforms. A gallery example
> never creates an exemption from those rules or from anti-slop/accessibility
> gates.

## Selection Rules

1. Start with Land-book for balanced common blocks and full-page sequence
   context.
2. Start with One Page Love when breadth or a conversion subtype matters,
   especially proof, pricing, FAQ, capture, CTA, mockups, and testimonials.
3. Start with Lapa for loaders, menus, hover, pointer interaction, galleries,
   footers, and text effects.
4. Use SiteInspire, Recent, Awwwards, or CSSDA when intentionally replacing
   conventional anatomy with a spatial canvas, proof object, cinematic state,
   or editorial index.
5. Distill recurring structure from several examples. Do not reproduce a
   single site's assets, copy, CSS, or signature composition; captured material
   follows `reference-capture.md`.
